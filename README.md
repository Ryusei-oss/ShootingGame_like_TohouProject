# 東方風弾幕シューティングゲーム（STG）

- 今までのシューティングゲームは被弾し続けるとゲームオーバーになってしまいます。実力のある人が長く遊べて、特に初心者の方々はすぐにやられてしまい長く遊ぶことができず実力を身に付けることが中々できません。
やられて上手くなれという考え方が新規参入を拒んでしまっています。そこで、本ゲームは最後まで遊べるSTGを作り、初心者がSTGゲームに興味を持つきっかけを作れたらと思います。

# 遊び方
- 敵の弾をできる限り避けて、最後のボスを倒してください
- 敵の弾に当たった回数が少ないほど上手です
- フォルダをダウンロードしたらコンソールでruby game.rbとコマンドを実行するとゲームが始まります
